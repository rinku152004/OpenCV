📘 R&D: Video Processing Pipeline Using Local Video
🔹 Objective

To design and implement a multi-threaded video processing pipeline using Python and OpenCV that reads a local video file, processes frames in stages, and displays results in real time.

🔹 System Design

The system follows a pipeline architecture, where each stage runs independently using threads and communicates via queues.

Video File
   ↓
Frame Reader
   ↓
Bilateral Filter
   ↓
Frame Crop
   ↓
Frame Difference
   ↓
Display Output
🔹 Key Concepts Used
1️⃣ Video Input
Uses cv2.VideoCapture
Reads frames sequentially until video ends
2️⃣ Multi-threading
Each stage runs in a separate thread
Improves performance and simulates real-time systems
3️⃣ Queues
Used for thread-safe communication
Each stage passes data to next stage
4️⃣ Image Processing
Bilateral Filter
Reduces noise
Preserves edges
Cropping
Removes bottom 10% of frame
Frame Difference
Uses:
cv2.absdiff(prev_frame, current_frame)
Detects motion/change
5️⃣ Display
Shows:
Original frame
Difference frame (side by side)
⚙️ Implementation Code
import cv2
import threading
import queue
import numpy as np

video_path = "video.mp4"

# Queues
q1 = queue.Queue(maxsize=10)
q2 = queue.Queue(maxsize=10)
q3 = queue.Queue(maxsize=10)
q4 = queue.Queue(maxsize=10)

stop_event = threading.Event()

# ---------- Reader ----------
def reader():
    cap = cv2.VideoCapture(video_path)

    while not stop_event.is_set():
        ret, frame = cap.read()
        if not ret:
            break
        q1.put(frame)

    q1.put(None)
    cap.release()

# ---------- Filter ----------
def filter_stage():
    while not stop_event.is_set():
        frame = q1.get()
        if frame is None:
            q2.put(None)
            break

        filtered = cv2.bilateralFilter(frame, 9, 75, 75)
        q2.put(filtered)

# ---------- Crop ----------
def crop_stage():
    while not stop_event.is_set():
        frame = q2.get()
        if frame is None:
            q3.put(None)
            break

        h = frame.shape[0]
        cropped = frame[:int(h * 0.9), :]
        q3.put(cropped)

# ---------- Difference ----------
def diff_stage():
    prev = None

    while not stop_event.is_set():
        frame = q3.get()
        if frame is None:
            q4.put(None)
            break

        if prev is None:
            prev = frame
            continue

        diff = cv2.absdiff(prev, frame)
        q4.put((frame, diff))
        prev = frame

# ---------- Display ----------
def display():
    while not stop_event.is_set():
        data = q4.get()
        if data is None:
            break

        frame, diff = data

        combined = np.hstack([frame, diff])
        combined = cv2.resize(combined, (800, 400))

        cv2.imshow("Pipeline Output", combined)

        if cv2.waitKey(1) == ord('q'):
            stop_event.set()
            break

    cv2.destroyAllWindows()

# ---------- Threads ----------
threads = [
    threading.Thread(target=reader),
    threading.Thread(target=filter_stage),
    threading.Thread(target=crop_stage),
    threading.Thread(target=diff_stage),
    threading.Thread(target=display)
]

for t in threads:
    t.start()

for t in threads:
    t.join()

print("Pipeline Finished ✅")
